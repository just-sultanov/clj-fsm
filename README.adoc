== clj-fsm

Simple finite state machine (FSM) in Clojure(Script)

=== Quick Start Guide

Add the following dependency in your project:

[source,clojure]
----
;; project.clj or build.boot
[clj-fsm "RELEASE"]

;; deps.edn
{:deps {clj-fsm {:mvn/version "RELEASE"}}}

----
